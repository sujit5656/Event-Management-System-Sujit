# Event Management System

## Project Overview

Event Management System is a web-based application developed using PHP, MySQL, Bootstrap, HTML, CSS and JavaScript.

The system allows customers to:

* Register/Login
* Book Events Online
* Make Payments
* Download Payment Slips
* Submit Feedback

Admin can:

* Manage Bookings
* Manage Events
* Manage Vendors
* Manage Feedback
* View Dashboard
